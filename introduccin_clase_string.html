Introducción Clase String

         

La clase String está orientada a manejar cadenas de caracteres.

Las cadenas son útiles para almacenar datos que se pueden representar en forma de texto. Algunas de las operaciones más utilizadas en cadenas son verificar su longitud length, para construirlas y concatenarlas usando operadores de cadena + y +=, verificando la existencia o ubicación de caracteres o sub-cadenas con indexof o extraer sub-cadenas con el método substring.
