Indicaciones para realizar actividad

Implementar en un formulario de Windows algunos métodos de la clase String para el manejo de cadenas

Instrucciones:

  1. Revisa el video sobre métodos de la clase String.
  2. Revisa el sitio web de Microsoft Docs para conocer la implementación de los diferentes métodos para el manejo de cadena.
  3. Crea un proyecto de Windows forms .NET nombrándolo como ProyString-[tu nombre].
  4. Diseña en el proyecto creado, un formulario según las instrucciones establecidas en la guía proporcionada por el facilitador.
  5. Comprime la carpeta del proyecto para posteriormente subirlo a plataforma.

Tipo de actividad: Individual
